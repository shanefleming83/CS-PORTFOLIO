Coding Concepts I Used

  • I used an "if then" variable to make a star move then change the screen and finally let the user see the message.
  • I used a loop to make the dog in the message look like it was excited and shaking.
  • I used a three sprites to make the message look better and more welcoming.

One Part of the Code I Understand Well

When the user presses the space key, my code makes the star spin and go down because of a counter feature that I added.

One Bug or Challenge I Fixed

For the first page transition to look good and to get rid of the star was hard because at first I set the thing at when the star reaches y=400 it would change but most of the star would still be visible so I had to spend a lot of time seeing how I could make it so that the whole star is not visible but the user isn't holding the key for too long.

One Thing I Would Improve

I would add more sprites to the final message.
