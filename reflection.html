Final Growth Reflection

1. What is one skill you improved the most in this class?

I think the one skill I improved most was just asking questions. Reaching out when I need help is usually a weak spot for me but this class really showed me that getting help is perfectly fine.

2. What is one project you are most proud of, and why?

I am most proud of my AI project, Organized. It was really tough figuring out how to make the AI think about timing and different catagories.

3. What is one project that did not go perfectly, but taught you something important?

My video game project was horrible. But it taught me that even if it is horrible it can still be liked by someone.

4. How did you get better at solving problems when something did not work?

I just started looking for help. When I got stumped I would either ask a friend for what they saw as a problem and then ask the teacher for further instruction.

5. How did this class change the way you think about technology, AI, the internet, or digital communication?

It mad me realise how intricate the system of the internet really is. It also made me rethink a lot of my passwords after our Cybersecurity unit.

6. What is one skill from this class that could help you in another class, job, hobby, or future career?

The skill of asking for help is the most important. It can represent the side of problem solving that most don't see and it would help me anywhere I have a team of coworkers.

Code and Design Defense

Choice 1

One design or code choice I made was changing the color of the two head colors at the top of each screen in the website. I made this choice because I wasn't keen on the default coloring. It helps the viewer by making the website appearance more appealing.

Choice 2

One design or code choice I made was the color of the text. I made this choice because black text for the headings is too boring. It helps the viewer by making the site more interesting and it makes it seem like more effort was put into the site.

Choice 3

One design or code choice I made was the font. I made this choice because Arial font is really basic and is just the default option. It helps the viewer by letting them know that there was a lot of detail put into the website.
